# AI Consulting Business Website

A modern, responsive website for AI consulting services, built with HTML, Tailwind CSS, and vanilla JavaScript.

## Project Structure

- `Nexus-AI-HTML/` - Contains all HTML pages
  - `index.html` - Homepage
  - `about.html` - About page
  - `services.html` - Services page
  - `case-studies.html` - Case studies
  - `careers.html` - Careers page
  - `methodology.html` - Methodology page
  - `audit.html` - Audit request page
  - `login.html` - Client login page
  - `main.html` - Main page

## Deployment

This project is configured for deployment on Vercel. The site is automatically deployed when changes are pushed to the main branch.

## Local Development

Simply open any HTML file in a web browser or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server Nexus-AI-HTML -p 8000
```

## Customization

Edit the HTML files in the `Nexus-AI-HTML/` directory to customize content, branding, and styling.
